## Dijkstra

## Présentation
Ce travail porte sur l'algorithme de Dijkstra.  
L'objectif est de trouver le plus court chemin dans un graphe pondéré à partir d'un sommet de départ jusqu'à un sommet d'arrivée.

## Contenu du dossier
- `graph.py` : contient la structure du graphe utilisée dans le travail
- `dijkstra_tp.py` : contient l'algorithme de Dijkstra et l'affichage du résultat
- `check_results.py` : permet de vérifier le chemin trouvé et la distance
- `RAPPORT.ipynb` : présente le travail avec explications, résultat et image du graphe
- `images/dijkstra_graph.png` : image du graphe utilisé dans le rapport

## But du travail
Le programme cherche le plus court chemin de `a` vers `e`.

## Résultat obtenu
- **Chemin le plus court :** `a -> c -> f -> e`
- **Distance totale :** `20`

## Exécution
Dans le terminal, exécuter les fichiers suivants :

```bash
python dijkstra_tp.py
python check_results.py
```
