📘 Projet : Figures Géométriques en Python

Nom : Maouchi Mohamed Chafik Numéro étudiant : 300154142

✨ Introduction

Dans ce projet, nous avons développé un programme en Python permettant de représenter et manipuler différentes figures géométriques en deux dimensions (2D) et en trois dimensions (3D).

Chaque figure est modélisée à l’aide d’une classe spécifique héritée d’une classe de base. Le programme permet de :

calculer automatiquement l’aire ou le volume afficher les informations de chaque figure visualiser graphiquement les formes avec la bibliothèque matplotlib 🎯 Objectifs du projet

Ce projet a pour but de :

mettre en pratique la programmation orientée objet (POO) utiliser l’héritage entre les classes automatiser les calculs géométriques représenter visuellement des figures en 2D et 3D mieux comprendre la géométrie dans un repère cartésien 🔹 Figures géométriques utilisées Figures en 2D

Les figures planes étudiées sont :

le carré (défini par la longueur de son côté) le cercle (défini par son rayon) le losange (défini par ses deux diagonales) le parallélogramme (défini par sa base et sa hauteur) Figures en 3D

Les solides étudiés sont :

le cylindre le prisme carré le prisme losange le prisme parallélogramme la sphère le cône 🧩 Organisation du projet

Le projet est structuré en plusieurs fichiers Python, chacun correspondant à une figure géométrique. Une classe de base permet de regrouper les caractéristiques communes, tandis que chaque classe spécifique implémente ses propres méthodes de calcul et d’affichage.

Des fonctions ont également été créées pour afficher graphiquement chaque figure à l’aide de matplotlib.

⚙️ Fonctionnement général

Pour chaque figure :

on crée un objet (ex : carré, cercle, etc.) on calcule son aire ou son volume on affiche ses informations on génère une représentation graphique

Cela permet de visualiser concrètement les formes tout en validant les calculs réalisés.

🎉 Conclusion

Ce projet m’a permis de mieux comprendre les principes fondamentaux de la programmation orientée objet en Python, notamment l’utilisation des classes et de l’héritage.

Il m’a également permis de travailler sur la représentation graphique des figures géométriques et d’automatiser les calculs mathématiques.
