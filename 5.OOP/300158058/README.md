## Description

Ce projet montre l’utilisation de la programmation orientée objet en Python avec quelques formes géométriques.

Les classes utilisées sont :

- `Figure` : classe de base
- `Carre` : classe dérivée
- `Cercle` : classe dérivée
- `Triangle` : classe dérivée
- `Cube` : classe ajoutée pour représenter une forme en 3D

Le programme principal crée plusieurs objets et affiche leurs informations.  
Cela permet de voir l’héritage, les méthodes et un exemple simple de polymorphisme.
