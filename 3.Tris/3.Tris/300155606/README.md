## Description du projet
Ce projet consiste à réaliser trois exercices sur les algorithmes de tri classiques :
- Tri par insertion
- Tri de Shell
- Tri rapide (Quick Sort)

L’objectif est de lire des données depuis un fichier texte, appliquer l’algorithme de tri correspondant, puis afficher le résultat trié.
