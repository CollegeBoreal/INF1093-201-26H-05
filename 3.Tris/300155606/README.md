## Description du projet
Ce projet consiste à réaliser trois algorithmes de tri :
- Tri par insertion
- Tri de Shell
- Tri rapide (Quick Sort)

Le programme lit des nombres depuis un fichier texte, les trie, puis affiche le résultat.
