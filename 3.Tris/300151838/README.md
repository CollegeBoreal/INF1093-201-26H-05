# 📘 Projet : Tri des données avec Python

Nom : Ryma Benhamouche  
ID : 300151838  
Cours : INF1093 – Structures de données  

---

## 🎯 But du projet

Ce projet consiste à lire des nombres depuis des fichiers texte, puis à les trier en ordre croissant avec différents algorithmes.

Les données sont séparées du code pour une meilleure organisation.

---

## 🧠 Algorithmes utilisés

- Tri par insertion  
- Tri de Shell  
- Tri rapide (Quick Sort)  

---

## 📂 Structure

insertion/
shell/
quick/

Chaque dossier contient un programme Python et un fichier texte.

---

## 📌 Conclusion

Ce projet permet de comprendre les algorithmes de tri et la gestion des fichiers en Python.
