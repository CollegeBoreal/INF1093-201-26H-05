# Algorithmes de tri – Lecture depuis fichiers

## Description
Ce projet implémente trois algorithmes de tri :
- Tri par insertion
- Tri de Shell
- Tri rapide (Quick Sort)

Les données sont lues depuis des fichiers texte contenant des entiers séparés par des espaces.

## Exécution

Se placer dans un dossier (ex: insertion/) puis exécuter :

python main.py
